This folder contains the code for the models utilized for task 1 - text classification

task1-word2vec, uses word2vec embeddings with ANN. 
task1-bert, uses bert embeddings with ANN.
task1-fasttext, uses fasttext embeddings with ANN.
task1-glove, uses glove embeddings with ANN.
task1-xlnet, uses xlnet/bert embeddings with ANN.

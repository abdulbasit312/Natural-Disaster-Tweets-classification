This folder contains the code for the models utilized for task 2 - text classification

task2-albert(lastlayer), uses ALbert word embeddings from the last layer passed to an ANN. 

task2-bert(concat), uses bert word embeddings from the last 4 concatenated layers passed to an ANN.
 
task2-distilbert(lastlayer), uses distilbert word embeddings from the last layer passed to an ANN.  

task2-distilgpt2(concat), uses distilgpt2 decoder architecture and uses the concatenated embeddings from last 4 layers passed to an ANN.  

task2-xlnet(lastlayer), uses xlnet word embeddings from the last layer passed to an ANN.
